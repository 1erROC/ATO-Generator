Génération conforme aux blocs USMTF (header + mission blocks). Target Type codé, PV CHAR codé (structure/pv/roof), IFF auto (octal).
IMPORT ATO
Colle ici un ATO issu de cet outil (ou un format proche). Le parseur extrait OPER/MSGID/TIMEFRAM et les blocs TASKUNIT/AMSNDAT/MSNACFT + set.
Note : si certains champs ne sont pas présents dans le texte importé, ils restent inchangés.
